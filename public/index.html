How to use the C++ REPL

  • Local Statements: Type commands (like int x = 42; or cout << x * 2 << endl;) directly in the terminal and hit Enter to run.
  • Variable Preservation: Variables declared in previous commands (e.g. string name = "Antigravity";) are fully accessible in subsequent commands.
  • Global Declarations: You can define helper functions and structs (e.g. struct User { string name; int age; };). They are compiled in the global namespace.
  • Header Includes: Type #include <set> to dynamically load new library headers into your session.
  • Script Editor: Write larger programs in the editor. Click Run Standalone to run the entire script (must have int main()) or Run in REPL to inject the editor script directly into the running session.
